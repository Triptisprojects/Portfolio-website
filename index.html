Developed an electronic voting (e-voting) system using Java, with Java frames employed for the development of the frontend. CPU Scheduling Algorithms

Designed and implemented programs to enact various CPU scheduling algorithms utilized by operating systems. These programs, developed proficiently in C++, perform comprehensive calculations of average waiting time for processes and their respective thresholds. Additionally, the software generates a visually informative Gantt chart based on input parameters including arrival time, burst time, and completion time of processes.
